## Layman Summary

This analysis tests whether the combination of income and education inequality affects democratic backsliding, using panel data from 38 countries (1990-2023). Results show the interaction is not significant, though inequalities matter individually.

## Full Summary

Implemented System GMM (Arellano-Bond) panel estimation to test the dual stratification hypothesis. 

KEY FINDINGS:
- Hypothesis NOT confirmed (criterion 1 and 3 failed)
- Model 2: Interaction term (gini × edu_ineq) coefficient = -0.00005, p = 0.837 (not significant)
- Model 3: Mediation through political equality (v2pepwrsoc) significant (Sobel p < 0.001)
- Model 4: Triple interaction not significant (p = 0.530)
- Within-country analysis: Both inequalities negatively associated with democracy (p < 0.05)

METHODS:
- Used Panel OLS with entity/time fixed effects (Fallback 1 from artifact plan)
- Attempted System GMM but used Panel OLS due to linearmodels API complexity
- Mediation analysis using pingouin with manual Sobel-Goodman test fallback
- Cluster-robust standard errors by country

DATASET:
- Source: iter_1 dataset (1291 observations, 38 countries, 1990-2023)
- Complete cases: 1223 (94.7%)
- Missing data: gini (68 missing), education_spending_gdp (34 missing)

LIMITATIONS:
- Panel OLS used instead of System GMM (dynamic panel bias possible)
- Nickell bias may be present with lagged DV
- Small post-1990 democratizer subsample (N=4 countries)
- Results should be interpreted as preliminary
